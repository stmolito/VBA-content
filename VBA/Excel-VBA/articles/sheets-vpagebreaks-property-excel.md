# Sheets.VPageBreaks Property (Excel)

Returns a  **[VPageBreaks](sheets-vpagebreaks-property-excel.md)** collection that represents the vertical page breaks on the sheet. Read-only.


## Syntax

 _expression_ . **VPageBreaks**

 _expression_ A variable that represents a **Sheets** object.
