# TickLabels.NumberFormat Property (Excel)

Returns or sets a  **String** value that represents the format code for the object.


## Syntax

 _expression_ . **NumberFormat**

 _expression_ A variable that represents a **TickLabels** object.


## Remarks

The format code is the same string as the  **Format Codes** option in the **Format Cells** dialog box. The **Format** function uses different format code strings than do the **NumberFormat** and **[NumberFormatLocal](ticklabels-numberformatlocal-property-excel.md)** properties.
