# ChartGroup.BinsOverflowEnabled Property (Excel)

Specifies whether a bin for values above the [BinsOverflowValue](chartgroup-binsoverflowvalue-property-excel.md) is enabled. Read/write **Boolean**.


## Syntax

 _expression_ . **BinsOverflowEnabled**

 _expression_ A variable that represents a **ChartGroup** object.
