# LegendKey.Delete Method (Excel)

Deletes the object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **LegendKey** object.


### Return Value

Variant


## Remarks

Deleting a  **[LegendKey](legendkey-object-excel.md)** object deletes the entire series.
