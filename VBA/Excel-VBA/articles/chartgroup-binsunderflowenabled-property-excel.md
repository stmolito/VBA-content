# ChartGroup.BinsUnderflowEnabled Property (Excel)

Specifies whether a bin for values below the [BinsUnderflowValue](chartgroup-binsunderflowvalue-property-excel.md) is enabled. Read/write **Boolean**.


## Syntax

 _expression_ . **BinsUnderflowEnabled**

 _expression_ A variable that represents a **ChartGroup** object.
