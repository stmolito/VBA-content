# SlicerCache.VisibleSlicerItemsList Property (Excel)

Returns or sets the list of MDX unique names for members at all levels of the hierarchy where manual filtering is applied. Read/write


## Syntax

 _expression_ . **VisibleSlicerItemsList**

 _expression_ A variable that represents a **[SlicerCache](slicercache-object-excel.md)** object.


### Return Value

 **Variant**


## Remarks

The  **VisibleSlicerItemsList** property is only applicable for slicers that are based on OLAP data sources ( **SlicerCache** . **[OLAP](slicercache-olap-property-excel.md)** = **True** ).
