# ProtectedViewWindow.Close Method (Excel)

Closes the specified  **Protected View** window.


## Syntax

 _expression_ . **Close**

 _expression_ A variable that represents a **[ProtectedViewWindow](protectedviewwindow-object-excel.md)** object.


### Return Value

Boolean
