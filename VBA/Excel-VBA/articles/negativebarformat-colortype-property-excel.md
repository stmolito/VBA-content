# NegativeBarFormat.ColorType Property (Excel)

 Specifies whether to use the same fill color as positive data bars. Read/write


## Syntax

 _expression_ . **ColorType**

 _expression_ A variable that represents a **[NegativeBarFormat](negativebarformat-object-excel.md)** object.


### Return Value

 **[XlDataBarNegativeColorType](xldatabarnegativecolortype-enumeration-excel.md)**


## Remarks

If you set the  **ColorType** property to **xlDataBarColor** , use the **[Color](negativebarformat-color-property-excel.md)** property of the **NegativeBarFormat** object to return a **[FormatColor](formatcolor-object-excel.md)** object that you can use to specify the fill color.
