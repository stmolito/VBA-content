# WorksheetView.Sheet Property (Excel)

Returns the sheet name for the specified  **[WorksheetView](worksheetview-object-excel.md)** object. Read-only.


## Syntax

 _expression_ . **Sheet**

 _expression_ A variable that represents a **WorksheetView** object.
