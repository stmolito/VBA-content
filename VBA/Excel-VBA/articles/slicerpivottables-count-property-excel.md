# SlicerPivotTables.Count Property (Excel)

Returns the number of objects in the collection.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that returns a **[SlicerPivotTables](slicerpivottables-object-excel.md)** object.


### Return Value

 **Long**
