# TickLabels.NumberFormatLocal Property (Excel)

Returns or sets a  **Variant** value that represents the format code for the object as a string in the language of the user.


## Syntax

 _expression_ . **NumberFormatLocal**

 _expression_ A variable that represents a **TickLabels** object.


## Remarks

The  **Format** function uses different format code strings than do the **[NumberFormat](ticklabels-numberformat-property-excel.md)** and **NumberFormatLocal** properties.
