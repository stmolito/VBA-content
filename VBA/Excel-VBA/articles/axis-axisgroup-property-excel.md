# Axis.AxisGroup Property (Excel)

Returns the group for the specified axis. Read-only


## Syntax

 _expression_ . **AxisGroup**

 _expression_ A variable that represents an **[Axis](axis-object-excel.md)** object.


### Return Value

 **[XlAxisGroup](xlaxisgroup-enumeration-excel.md)**
