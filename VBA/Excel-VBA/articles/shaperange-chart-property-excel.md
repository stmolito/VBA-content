# ShapeRange.Chart Property (Excel)

Returns a  **[Chart](chart-object-excel.md)** object that represents the chart contained in the shape range. Read-only.


## Syntax

 _expression_ . **Chart**

 _expression_ A variable that represents a **ShapeRange** object.
