# FileExportConverters.Application Property (Excel)

Returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[FileExportConverters](fileexportconverters-object-excel.md)** collection.
