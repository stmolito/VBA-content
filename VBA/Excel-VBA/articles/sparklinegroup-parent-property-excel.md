# SparklineGroup.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[SparklineGroup](sparklinegroup-object-excel.md)** object.
