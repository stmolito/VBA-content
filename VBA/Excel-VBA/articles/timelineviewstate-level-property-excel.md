# TimelineViewState.Level Property (Excel)

Get or set the displayed level. [XlTimelineLevel Enumeration (Excel)](xltimelinelevel-enumeration-excel.md) Read/Write


## Syntax

 _expression_ . **Level**

 _expression_ A variable that represents a[TimelineViewState](timelineviewstate-object-excel.md) object.


## Property value

 **XLTIMELINELEVEL**
