# SparklineGroup.Count Property (Excel)

Returns the number of sparklines in the sparkline group. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[SparklineGroup](sparklinegroup-object-excel.md)** object.


### Return Value

Long
