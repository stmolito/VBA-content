# RectangularGradient.RectangleBottom Property (Excel)

Represents the point or vector that the gradient fill converges to. Read/write


## Syntax

 _expression_ . **RectangleBottom**

 _expression_ A variable that represents a **RectangularGradient** object.


### Return Value

Double


## Remarks

Works with RectangleLeft, RectangleRight, and RectangleTop. Valid values are listed in the table below.



|**Property**|**Values**|
|:-----|:-----|
|[RectangleLeft](rectangulargradient-rectangleleft-property-excel.md)|0-1|
|[RectangleRight](rectangulargradient-rectangleright-property-excel.md)|0-1|
|[RectangleTop](rectangulargradient-rectangletop-property-excel.md)|0-1|
|RectangleBottom|0-1|
