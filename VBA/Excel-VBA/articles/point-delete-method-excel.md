# Point.Delete Method (Excel)

Deletes the series the point belongs to.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **Point** object.


### Return Value

Variant


## Remarks

Deleting a  **[Point](point-object-excel.md)** object deletes the entire series.
