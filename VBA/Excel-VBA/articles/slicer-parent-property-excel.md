# Slicer.Parent Property (Excel)

Returns the  **[Worksheet](worksheet-object-excel.md)** object that represents the sheet that contains the slicer. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **Slicer** object.
