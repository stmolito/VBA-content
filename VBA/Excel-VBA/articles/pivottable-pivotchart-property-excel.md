# PivotTable.PivotChart Property (Excel)

Returns a [Shape Object (Excel)](shape-object-excel.md) object that represents the standalone PivotChart for the specified hidden PivotTable report. Read-only.


## Syntax

 _expression_ . **PivotChart**

 _expression_ A variable that represents a[PivotTable Object (Excel)](pivottable-object-excel.md) object.


## Returns

[Shape Object (Excel)](shape-object-excel.md)


## Property value

 **SHAPE**
