# ChartGroup.CategoryCollection Method (Excel)

Returns an object that represents a collection of all of the visible categories (a [CategoryCollection](categorycollection-object-excel.md) collection) in the chart group.


## Syntax

 _expression_ . **CategoryCollection**_(Index)_

 _expression_ A variable that represents a **ChartGroup** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Optional|VARIANT|The name or number of the categories.|

### Return value

 **OBJECT**
