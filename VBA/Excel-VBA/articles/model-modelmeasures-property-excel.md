# Model.ModelMeasures Property (Excel)

Returns a [ModelMeasures Object (Excel)](modelmeasures-object-excel.md) object that represents the collection of model measures in the data model. Read-only.


## Syntax

 _expression_ . **ModelMeasures**

 _expression_ A variable that represents a **Model** object.
