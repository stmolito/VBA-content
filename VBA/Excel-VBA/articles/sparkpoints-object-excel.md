# SparkPoints Object (Excel)

Represents the settings of the markers for points of data on a sparkline.


## Remarks

Use the  **SparkPoints** object to set the color and visibility of markers for points of data on a sparkline. Use the **[Points](http://msdn.microsoft.com/library/afcc972f-aa1b-74f8-7b00-332cc3fa17a3%28Office.15%29.aspx)** property of the **[SparklineGroup](sparklinegroup-object-excel.md)** object to return a **SparkPoints** object. The properties of the **SparkPoints** object correspond to the settings of the **High Point**,  **Low Point**,  **Negative Point**,  **First Point**,  **Last Point**, and  **Markers** check boxes in the **Show** section, and of the items on the **Marker Color** drop-down list in the **Style** section of the ** Sparkline Tools Design** tab of the ribbon.
