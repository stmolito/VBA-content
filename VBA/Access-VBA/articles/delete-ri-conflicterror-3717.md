# Delete/RI conflict. (Error 3717)

  

**Applies to:** Access 2013 | Access 2016

The primary key record has been deleted by another replica, therefore this referencing record has been rejected. Either create a new primary key that satisfies the referential integrity constraint and then resubmit your update, or delete this conflict record.
