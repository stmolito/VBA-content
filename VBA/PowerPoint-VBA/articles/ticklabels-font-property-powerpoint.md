# TickLabels.Font Property (PowerPoint)

Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-powerpoint.md)**.


## Syntax

 _expression_. **Font**

 _expression_ A variable that represents a **[TickLabels](ticklabels-object-powerpoint.md)** object.
