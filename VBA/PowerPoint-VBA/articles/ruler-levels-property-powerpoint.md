# Ruler.Levels Property (PowerPoint)

Returns a  **[RulerLevels](rulerlevels-object-powerpoint.md)** object that represents outline indent formatting. Read-only.


## Syntax

 _expression_. **Levels**

 _expression_ A variable that represents a **Ruler** object.
