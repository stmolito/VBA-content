Returns the type of axis group. Read/write  **[XlAxisGroup](xlaxisgroup-enumeration-powerpoint.md)**.


## Syntax

 _expression_. **AxisGroup**

 _expression_ A variable that represents a **[Series](series-object-powerpoint.md)** object.
