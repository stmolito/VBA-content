# FileConverter.Path Property (PowerPoint)

Returns the disk or Web path to the specified object. Read-only  **String**.


## Syntax

 _expression_. **Path**

 _expression_ Required. A variable that represents a **[FileConverter](fileconverter-object-powerpoint.md)** object.
