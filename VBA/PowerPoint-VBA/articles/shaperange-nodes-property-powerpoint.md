# ShapeRange.Nodes Property (PowerPoint)

Returns a  **[ShapeNodes](shapenodes-object-powerpoint.md)** collection that represents the geometric description of the specified shape. Applies to **ShapeRange** objects that represent freeform drawings.


## Syntax

 _expression_. **Nodes**

 _expression_ A variable that represents a **ShapeRange** object.
