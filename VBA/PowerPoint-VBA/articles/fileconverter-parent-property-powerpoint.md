# FileConverter.Parent Property (PowerPoint)

Returns an  **Object** that represents the parent object of the specified **[FileConverter](fileconverter-object-powerpoint.md)** object.


## Syntax

 _expression_. **Parent**

 _expression_ Required. A variable that represents a **[FileConverter](fileconverter-object-powerpoint.md)** object.
