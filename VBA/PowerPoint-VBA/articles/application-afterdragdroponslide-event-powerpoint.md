# Application.AfterDragDropOnSlide Event (PowerPoint)

Occurs after a shape has been dropped onto a slide in an open presentation.


## Syntax

 _expression_. **AfterDragDropOnSlide**_(Sld,_ _X,_ _Y)_

 _expression_ A variable that represents a **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|||||
| _Sld_|Required|[Slide](slide-object-powerpoint.md)|The slide that raised the event (that is, had a shape added to it).|
| _X_|Required|**Single**||
| _Y_|Required|**Single**||
| _Sld_|Required|SLIDE||
| _X_|Required|FLOAT||
| _Y_|Required|FLOAT||
