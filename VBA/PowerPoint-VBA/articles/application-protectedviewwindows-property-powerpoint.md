# Application.ProtectedViewWindows Property (PowerPoint)

Returns a  **[ProtectedViewWindows](protectedviewwindows-object-powerpoint.md)** collection that represents all the **Protected View** windows that are open in the application. Read-only


## Syntax

 _expression_. **ProtectedViewWindows**

 _expression_ A variable that represents an **Application** object.
