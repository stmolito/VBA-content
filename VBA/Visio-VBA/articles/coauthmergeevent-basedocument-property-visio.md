# CoauthMergeEvent.BaseDocument Property (Visio)

Returns a [Document](document-object-visio.md) object that represents the state of the original document before any changes by any users are merged into it. Read-only.


## Syntax

 _expression_ . **BaseDocument**

 _expression_ A variable that represents a **CoauthMergeEvent** object.


## Property value

 **IVDOCUMENT**
