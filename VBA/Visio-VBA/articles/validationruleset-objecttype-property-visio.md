# ValidationRuleSet.ObjectType Property (Visio)

Returns  **visObjTypeValidationRuleSet** , the type of this object. Read-only.


## Syntax

 _expression_ . **ObjectType**

 _expression_ A variable that represents a **[ValidationRuleSet](validationruleset-object-visio.md)** object.


### Return Value

 **visObjTypeValidationRuleSet**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](visobjecttypes-enumeration-visio.md)** enumeration.
