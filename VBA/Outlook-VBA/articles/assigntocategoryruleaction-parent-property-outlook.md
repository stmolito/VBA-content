# AssignToCategoryRuleAction.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **AssignToCategoryRuleAction** object.


## Remarks

The parent of the  **[AssignToCategoryRuleAction](assigntocategoryruleaction-object-outlook.md)** object is the **[RuleAction](ruleaction-object-outlook.md)** object.
