# ContactsModule.NavigationGroups Property (Outlook)

Returns a  **[NavigationGroups](navigationgroups-object-outlook.md)** object that contains the set of navigation groups associated with the **[ContactsModule](contactsmodule-object-outlook.md)** object. Read-only.


## Syntax

 _expression_ . **NavigationGroups**

 _expression_ A variable that represents a **ContactsModule** object.
