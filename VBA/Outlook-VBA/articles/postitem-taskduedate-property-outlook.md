# PostItem.TaskDueDate Property (Outlook)

Returns or sets a  **Date** value that represents the due date of the task for this **[PostItem](postitem-object-outlook.md)** . Read/write.


## Syntax

 _expression_ . **TaskDueDate**

 _expression_ An expression that returns a **PostItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the **[IsMarkedAsTask](postitem-ismarkedastask-property-outlook.md)** property is set to **False** .
