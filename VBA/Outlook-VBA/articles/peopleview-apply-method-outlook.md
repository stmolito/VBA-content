# PeopleView.Apply Method (Outlook)
Applies the [PeopleView](peopleview-object-outlook.md) object to the current view.

## Version information

Version Added: Outlook 2013 


## Syntax

 _expression_ . **Apply**

 _expression_ A variable that represents a **PeopleView** object.
