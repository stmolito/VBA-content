# SharingItem.ConversationTopic Property (Outlook)

Returns a  **String** representing the topic of the conversation thread of the **[SharingItem](sharingitem-object-outlook.md)** . Read-only.


## Syntax

 _expression_ . **ConversationTopic**

 _expression_ A variable that represents a **SharingItem** object.
