# JournalModule.Class Property (Outlook)

Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **JournalModule** object.


## Remarks

This property returns  **olJournalModule** for the **[JournalModule](journalmodule-object-outlook.md)** object.
