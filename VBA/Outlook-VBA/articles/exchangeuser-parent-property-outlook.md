# ExchangeUser.Parent Property (Outlook)

Returns the parent  **Object** of the **[ExchangeUser](exchangeuser-object-outlook.md)** object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **ExchangeUser** object.
