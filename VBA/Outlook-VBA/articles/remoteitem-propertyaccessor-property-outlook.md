# RemoteItem.PropertyAccessor Property (Outlook)

Returns a  **[PropertyAccessor](propertyaccessor-object-outlook.md)** object that supports creating, getting, setting, and deleting properties of the parent **[RemoteItem](remoteitem-object-outlook.md)** object. Read-only.


## Syntax

 _expression_ . **PropertyAccessor**

 _expression_ A variable that represents a **RemoteItem** object.
