# Table.Class Property (Outlook)

Returns a constant in the  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** enumeration indicating the class of the **[Table](table-object-outlook.md)** object. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **Table** object.
