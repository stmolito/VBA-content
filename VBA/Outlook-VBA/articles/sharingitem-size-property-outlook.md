# SharingItem.Size Property (Outlook)

Returns a  **Long** indicating the size (in bytes) of the **[SharingItem](sharingitem-object-outlook.md)** . Read-only.


## Syntax

 _expression_ . **Size**

 _expression_ A variable that represents a **SharingItem** object.
