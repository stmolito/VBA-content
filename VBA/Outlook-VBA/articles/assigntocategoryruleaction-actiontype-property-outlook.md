# AssignToCategoryRuleAction.ActionType Property (Outlook)

Returns a constant,  **olRuleActionAssignToCategory** , from the **[OlRuleActionType](olruleactiontype-enumeration-outlook.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.


## Syntax

 _expression_ . **ActionType**

 _expression_ A variable that represents an **AssignToCategoryRuleAction** object.


## Remarks

 The category assigning rule action is supported by rules for receiving messages and rules for sending messages. For more information, see[Specifying Rule Actions](http://msdn.microsoft.com/library/c5f83c81-0e01-38aa-5ec7-3932b4443e43%28Office.15%29.aspx).
