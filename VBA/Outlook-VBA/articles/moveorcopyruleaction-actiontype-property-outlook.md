# MoveOrCopyRuleAction.ActionType Property (Outlook)

Returns a constant,  **olRuleActionMoveToFolder** , from the **[OlRuleActionType](olruleactiontype-enumeration-outlook.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.


## Syntax

 _expression_ . **ActionType**

 _expression_ A variable that represents a **MoveOrCopyRuleAction** object.


## Remarks

 The moving or copying rule action is supported only by rules for receiving messages. For more information, see[Specify Rule Actions](http://msdn.microsoft.com/library/c5f83c81-0e01-38aa-5ec7-3932b4443e43%28Office.15%29.aspx).
