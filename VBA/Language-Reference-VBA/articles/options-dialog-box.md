# Options Dialog Box


![Options dialog box](images/opdlvbe_ZA01201635.gif)



Allows you to change default settings for the Visual Basic development environment.

The Options dialog box has the following tabs:



- Editor
    
- Editor Format
    
- General
    
- Docking
