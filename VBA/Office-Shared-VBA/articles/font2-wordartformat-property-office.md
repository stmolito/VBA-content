# Font2.WordArtformat Property (Office)

Gets or sets a value specifying the text effect for the selected text. Read/write.


## Syntax

 _expression_. **WordArtformat**

 _expression_ An expression that returns a **Font2** object.


### Return Value

MsoPresetTextEffect
