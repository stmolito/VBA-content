# FileDialogFilters.Count Property (Office)

Gets a  **Long** indicating the number of items in the **FileDialogFilters** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **FileDialogFilters** object.


### Return Value

Long
