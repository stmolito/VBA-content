# MetaProperty.Type Property (Office)

Gets the data type of a  **MetaProperty** object. Read-only.


## Syntax

 _expression_. **Type**

 _expression_ An expression that returns a **MetaProperty** object.


### Return Value

MsoMetaPropertyType
