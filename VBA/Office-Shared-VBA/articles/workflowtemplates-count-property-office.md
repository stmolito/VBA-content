# WorkflowTemplates.Count Property (Office)

Gets a  **Long** indicating the number of items in the **WorkflowTemplates** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **WorkflowTemplates** object.


### Return Value

Long
