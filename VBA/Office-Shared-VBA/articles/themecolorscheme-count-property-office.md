# ThemeColorScheme.Count Property (Office)

Gets a  **Integer** indicating the number of items in the **ThemeColorScheme** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **ThemeColorScheme** object.


### Return Value

Integer
