# CustomXMLSchemaCollection.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLSchemaCollection** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLSchemaCollection** object.


### Return Value

Object
