# CustomXMLValidationErrors.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLValidationErrors** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLValidationErrors** object.


### Return Value

Object
