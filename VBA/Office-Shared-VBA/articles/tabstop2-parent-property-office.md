# TabStop2.Parent Property (Office)

Gets an Object that represents the  **Parent** object for the **TabStop2** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **TabStop2** object.


### Return Value

Object
