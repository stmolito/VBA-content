# MsoEnvelope.Parent Property (Office)

Gets the  **Parent** object for the MsoEnvelope object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **MsoEnvelope** object.


### Return Value

Object
