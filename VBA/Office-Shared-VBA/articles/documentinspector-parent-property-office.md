# DocumentInspector.Parent Property (Office)

Gets an object that represents the parent of the  **DocumentInspector** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **DocumentInspector** object.


### Return Value

Object
