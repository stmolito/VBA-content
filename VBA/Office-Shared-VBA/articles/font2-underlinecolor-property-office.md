# Font2.UnderlineColor Property (Office)

Gets a value specifying the color of the underline for the selected text. Read-only.


## Syntax

 _expression_. **UnderlineColor**

 _expression_ An expression that returns a **Font2** object.


### Return Value

ColorFormat
