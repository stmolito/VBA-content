# FileDialogFilters.Parent Property (Office)

Gets the  **Parent** object for the **FileDialogFilters** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **FileDialogFilters** object.


### Return Value

Object
