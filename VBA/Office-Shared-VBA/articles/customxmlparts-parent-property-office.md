# CustomXMLParts.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLParts** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLParts** object.


### Return Value

Object
