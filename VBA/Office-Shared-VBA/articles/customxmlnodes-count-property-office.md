# CustomXMLNodes.Count Property (Office)

Gets a count of the number of  **CustomXMLNode** objects in a **CustomXMLNodes** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **CustomXMLNodes** object.


### Return Value

Long
