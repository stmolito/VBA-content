# ScopeFolders.Count Property (Office)

Gets a  **Long** indicating the number of items in the **ScopeFolders** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **ScopeFolders** object.


### Return Value

Long
