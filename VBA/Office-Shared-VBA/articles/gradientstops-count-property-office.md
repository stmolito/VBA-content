# GradientStops.Count Property (Office)

Gets a  **Long** indicating the number of items in the **GradientStops** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **GradientStops** object.


### Return Value

Integer
