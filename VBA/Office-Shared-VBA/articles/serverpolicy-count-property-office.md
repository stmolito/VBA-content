# ServerPolicy.Count Property (Office)

Gets a  **Long** indicating the number of items in the **ServerPolicy** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **ServerPolicy** object.


### Return Value

Long
