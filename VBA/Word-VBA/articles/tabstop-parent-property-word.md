# TabStop.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TabStop** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TabStop](tabstop-object-word.md)** object.
