# ListEntries.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ListEntries** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ListEntries](listentries-object-word.md)** collection.
