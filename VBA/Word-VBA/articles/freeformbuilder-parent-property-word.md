# FreeformBuilder.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FreeformBuilder** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[FreeformBuilder](freeformbuilder-object-word.md)** object.
