# OMathNary.Sup Property (Word)

Returns an  **[OMath](omath-object-word.md)** object that represents the upper limit of an n-ary operator. Read-only.


## Syntax

 _expression_ . **Sup**

 _expression_ An expression that returns an **[OMathNary](omathnary-object-word.md)** object.
