# OMathMatRows.Count Property (Word)

Returns the number of items in the  **OMathMatRows** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[OMathMatRows](omathmatrows-object-word.md)** object.
