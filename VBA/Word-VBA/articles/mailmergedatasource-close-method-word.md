# MailMergeDataSource.Close Method (Word)

Closes the specified Mail Merge data source.


## Syntax

 _expression_ . **Close**

 _expression_ Required. A variable that represents a **[MailMergeDataSource](mailmergedatasource-object-word.md)** object.
