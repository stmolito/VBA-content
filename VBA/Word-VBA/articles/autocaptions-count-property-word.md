# AutoCaptions.Count Property (Word)

Returns the number of items in the  **AutoCaptions** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[AutoCaptions](autocaptions-object-word.md)** object.
