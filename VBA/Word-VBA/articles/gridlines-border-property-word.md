# Gridlines.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-word.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents a **[GridLines](gridlines-object-word.md)** object.
