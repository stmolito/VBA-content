# DropDown.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **DropDown** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[DropDown](dropdown-object-word.md)** object.
