# MappedDataFields.Item Method (Word)

Returns an individual  **MappedDataField** object.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[MappedDataFields](mappeddatafields-object-word.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **WdMappedDataFields**|The specified mapped data field.|

### Return Value

MappedDataField
