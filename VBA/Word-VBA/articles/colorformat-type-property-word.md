# ColorFormat.Type Property (Word)

Returns or sets the shape color type. Read-only  **MsoColorType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[ColorFormat](colorformat-object-word.md)** object.
