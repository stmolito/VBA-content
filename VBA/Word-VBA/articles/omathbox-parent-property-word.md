# OMathBox.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathBox** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathBox](omathbox-object-word.md)** object.
