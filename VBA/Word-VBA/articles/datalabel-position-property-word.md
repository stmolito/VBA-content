# DataLabel.Position Property (Word)

Returns or sets the position of the data label. Read/write  **[XlDataLabelPosition](xldatalabelposition-enumeration-word.md)** .


## Syntax

 _expression_ . **Position**

 _expression_ A variable that represents a **[DataLabel](datalabel-object-word.md)** object.
