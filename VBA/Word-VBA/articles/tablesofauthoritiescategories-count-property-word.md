# TablesOfAuthoritiesCategories.Count Property (Word)

Returns a  **Long** that represents the number of categories in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[TablesOfAuthoritiesCategories](tablesofauthoritiescategories-object-word.md)** collection.
