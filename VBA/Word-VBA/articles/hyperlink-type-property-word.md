# Hyperlink.Type Property (Word)

Returns the hyperlink type. Read-only  **MsoHyperlinkType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[Hyperlink](hyperlink-object-word.md)** object.
