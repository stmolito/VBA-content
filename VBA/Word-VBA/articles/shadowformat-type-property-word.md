# ShadowFormat.Type Property (Word)

Returns or sets the shape shadow type. Read/write  **MsoShadowType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[ShadowFormat](shadowformat-object-word.md)** object.
