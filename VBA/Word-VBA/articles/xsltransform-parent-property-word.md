# XSLTransform.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **XSLTransform** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[XSLTransform](xsltransform-object-word.md)** object.
