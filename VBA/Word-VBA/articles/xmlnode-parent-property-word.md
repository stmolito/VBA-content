# XMLNode.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **XMLNode** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[XMLNode](xmlnode-object-word.md)** object.
