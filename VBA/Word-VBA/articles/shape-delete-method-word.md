# Shape.Delete Method (Word)

Deletes the specified shape node.


## Syntax

 _expression_ . **Delete**( **_Index_** )

 _expression_ Required. A variable that represents a **[Shape](shape-object-word.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The number of the shape node to delete.|
