# TablesOfAuthoritiesCategories.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TablesOfAuthoritiesCategories** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TablesOfAuthoritiesCategories](tablesofauthoritiescategories-object-word.md)** collection.
