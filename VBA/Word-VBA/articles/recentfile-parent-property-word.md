# RecentFile.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **RecentFile** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[RecentFile](recentfile-object-word.md)** object.
