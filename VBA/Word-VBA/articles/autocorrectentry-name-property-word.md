# AutoCorrectEntry.Name Property (Word)

Returns or sets the name of the specified object. Read/write  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents an **[AutoCorrectEntry](autocorrectentry-object-word.md)** object.
