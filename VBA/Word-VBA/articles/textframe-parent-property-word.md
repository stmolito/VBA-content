# TextFrame.Parent Property (Word)

Returns a  **Shape** object that represents the parent shape of the text frame.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TextFrame](textframe-object-word.md)** object.
