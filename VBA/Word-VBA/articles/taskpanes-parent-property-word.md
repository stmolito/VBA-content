# TaskPanes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TaskPanes** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TaskPanes](taskpanes-object-word.md)** collection.
