# XMLNode.Cut Method (Word)

Removes the specified XML element from the document and places it on the Clipboard.


## Syntax

 _expression_ . **Cut**

 _expression_ Required. A variable that represents a **[XMLNode](xmlnode-object-word.md)** object.
