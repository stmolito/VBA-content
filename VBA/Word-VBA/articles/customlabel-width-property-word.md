# CustomLabel.Width Property (Word)

Returns or sets the width of a custom mailing label, in points. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[CustomLabel](customlabel-object-word.md)** object.
