# DisplayUnitLabel.Position Property (Word)

Returns or sets the position of the unit label on an axis in the chart. Read/write  **[XlChartElementPosition](xlchartelementposition-enumeration-word.md)** .


## Syntax

 _expression_ . **Position**

 _expression_ A variable that represents a **[DisplayUnitLabel](displayunitlabel-object-word.md)** object.
