# DefaultWebOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **DefaultWebOptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[DefaultWebOptions](defaultweboptions-object-word.md)** collection.
