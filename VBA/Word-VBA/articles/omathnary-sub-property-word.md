# OMathNary.Sub Property (Word)

Returns an  **[OMath](omath-object-word.md)** object that represents the lower limit of an n-ary operator. Read-only.


## Syntax

 _expression_ . **Sub**

 _expression_ An expression that returns an **[OMathNary](omathnary-object-word.md)** object.
