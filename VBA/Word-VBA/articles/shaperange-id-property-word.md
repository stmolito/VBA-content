# ShapeRange.ID Property (Word)

Returns the identification type for the range of shapes. Read-only  **Long** .


## Syntax

 _expression_ . **ID**

 _expression_ Required. A variable that represents a **[ShapeRange](shaperange-object-word.md)** object.
