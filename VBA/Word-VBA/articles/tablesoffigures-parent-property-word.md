# TablesOfFigures.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TablesOfFigures** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TablesOfFigures](tablesoffigures-object-word.md)** collection.
