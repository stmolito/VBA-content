# Shapes.Count Property (Word)

Returns a  **Long** that represents the number of shapes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Shapes](shapes-object-word.md)** collection.
