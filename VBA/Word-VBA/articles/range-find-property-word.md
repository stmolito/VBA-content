# Range.Find Property (Word)

Returns a  **[Find](find-object-word.md)** object that contains the criteria for a find operation. Read-only.


## Syntax

 _expression_ . **Find**

 _expression_ A variable that represents a **[Range](range-object-word.md)** object.
