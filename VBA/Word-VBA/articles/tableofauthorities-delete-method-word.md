# TableOfAuthorities.Delete Method (Word)

Deletes the specified table of authorities.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[TableOfAuthorities](tableofauthorities-object-word.md)** collection.
