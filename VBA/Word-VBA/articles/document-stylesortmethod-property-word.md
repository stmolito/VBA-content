# Document.StyleSortMethod Property (Word)

Returns or sets a **[WdStyleSort](wdstylesort-enumeration-word.md)** constant that represents the sort method to use when sorting styles in the **Styles** task pane. Read/write.


## Syntax

 _expression_ . **StyleSortMethod**

 _expression_ An expression that returns a **[Document](document-object-word.md)** object.
