# HangulHanjaConversionDictionaries.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HangulHanjaConversionDictionaries** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HangulHanjaConversionDictionaries](hangulhanjaconversiondictionaries-object-word.md)** collection.
