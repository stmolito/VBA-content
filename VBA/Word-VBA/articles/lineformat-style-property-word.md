# LineFormat.Style Property (Word)

Returns or sets the line format style. Read/write  **MsoLineStyle** .


## Syntax

 _expression_ . **Style**

 _expression_ Required. A variable that represents a **[LineFormat](lineformat-object-word.md)** object.
