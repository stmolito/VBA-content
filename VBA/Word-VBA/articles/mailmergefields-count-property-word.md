# MailMergeFields.Count Property (Word)

Returns a  **Long** that represents the number of mail merge fields in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[MailMergeFields](mailmergefields-object-word.md)** collection.
