# UndoRecord.Application Property (Word)

Returns an [Application](application-object-word.md) object that represents the Word application. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents an **[UndoRecord](undorecord-object-word.md)** object.
