# Tasks.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Tasks** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Tasks](tasks-object-word.md)** collection.
