# Chart.SeriesNameLevel Property (Word)

Returns or sets an [XlSeriesNameLevel](xlseriesnamelevel-enumeration-word.md) constant that specifies the source level of the series names. Read-write.


## Syntax

 _expression_ . **SeriesNameLevel**

 _expression_ A variable that represents a **Chart** object.


## Property value

 **XLSERIESNAMELEVEL**
