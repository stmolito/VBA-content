# LegendKey.Smooth Property (Word)

 **True** if curve smoothing is turned on for the legend key. Read/write **Boolean** .


## Syntax

 _expression_ . **Smooth**

 _expression_ A variable that represents a **[LegendKey](legendkey-object-word.md)** object.
