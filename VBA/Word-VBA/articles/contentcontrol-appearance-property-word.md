# ContentControl.Appearance Property (Word)

Returns or sets the appearance of the content control. Read/write [WdContentControlAppearance](wdcontentcontrolappearance-enumeration-word.md).


## Syntax

 _expression_ . **Appearance**

 _expression_ A variable that represents a **ContentControl** object.


## Property value

 **WDCONTENTCONTROLAPPEARANCE**
