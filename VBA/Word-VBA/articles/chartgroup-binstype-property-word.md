# ChartGroup.BinsType Property (Word)

Specifies how the horizontal axis of the histogram chart is formatted, by bins type. Read/write [XlBinsType](xlbinstype-enumeration-word.md).


## Syntax

 _expression_ . **BinsType**

 _expression_ A variable that represents a **ChartGroup** object.
