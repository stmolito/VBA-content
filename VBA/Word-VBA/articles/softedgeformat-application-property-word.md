# SoftEdgeFormat.Application Property (Word)

Returns an [Application](application-object-word.md) object that represents the Word application. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[SoftEdgeFormat](softedgeformat-object-word.md)** object.
