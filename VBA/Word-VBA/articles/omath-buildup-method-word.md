# OMath.BuildUp Method (Word)

Converts an equation to professional format.


## Syntax

 _expression_ . **BuildUp**

 _expression_ An expression that returns an **OMath** object.


### Return Value

Nothing


## Remarks

To change an equation to linear format, use the  **[Linearize](omath-linearize-method-word.md)** method.
