# OMathFunc.FName Property (Word)

Returns an  **[OMath](omath-object-word.md)** object that represents the name of a mathematical function, such as sin or cos. Read-only.


## Syntax

 _expression_ . **FName**

 _expression_ An expression that returns an **[OMathFunc](omathfunc-object-word.md)** object.
