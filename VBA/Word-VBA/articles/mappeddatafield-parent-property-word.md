# MappedDataField.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MappedDataField** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[MappedDataField](mappeddatafield-object-word.md)** object.
