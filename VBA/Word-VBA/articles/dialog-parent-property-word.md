# Dialog.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Dialog** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Dialog](dialog-object-word.md)** object.
