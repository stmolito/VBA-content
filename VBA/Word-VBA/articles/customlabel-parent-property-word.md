# CustomLabel.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CustomLabel** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[CustomLabel](customlabel-object-word.md)** object.
