# AddIns.Count Property (Word)

Returns the number of  **[AddIn](addin-object-word.md)** objects in the **AddIns** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ An expression that returns an **[AddIns](addins-object-word.md)** collection.
