# MailMergeField.Next Property (Word)

Returns the next mail merge field in the collection of mail merge fields. Read-only.


## Syntax

 _expression_ . **Next**

 _expression_ A variable that represents a **[MailMergeField](mailmergefield-object-word.md)** object.
