# SpellingSuggestions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **SpellingSuggestions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[SpellingSuggestions](spellingsuggestions-object-word.md)** collection.
