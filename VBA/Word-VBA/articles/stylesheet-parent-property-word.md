# StyleSheet.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **StyleSheet** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[StyleSheet](stylesheet-object-word.md)** object.
