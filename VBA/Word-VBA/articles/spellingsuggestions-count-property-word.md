# SpellingSuggestions.Count Property (Word)

Returns a  **Long** that represents the number of spelling suggestions in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[SpellingSuggestions](spellingsuggestions-object-word.md)** collection.
