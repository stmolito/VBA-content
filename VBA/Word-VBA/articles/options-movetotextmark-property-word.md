# Options.MoveToTextMark Property (Word)

 Returns or sets a **[WdMoveToTextMark](wdmovetotextmark-enumeration-word.md)** constant that represents the type of revision mark to use for moved text. Read/write.


## Syntax

 _expression_ . **MoveToTextMark**

 _expression_ An expression that returns an **[Options](options-object-word.md)** object.
