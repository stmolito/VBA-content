# DataLabel.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **[DataLabel](datalabel-object-word.md)** object.
