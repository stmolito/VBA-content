# HorizontalLineFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HorizontalLineFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HorizontalLineFormat](horizontallineformat-object-word.md)** object.
