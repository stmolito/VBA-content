# WebOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **WebOptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[WebOptions](weboptions-object-word.md)** collection.
