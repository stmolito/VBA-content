# DropCap.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **DropCap** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[DropCap](dropcap-object-word.md)** object.
