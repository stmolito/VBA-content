# LinkFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **LinkFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[LinkFormat](linkformat-object-word.md)** object.
