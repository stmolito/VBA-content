# ThreeDFormat.Z Property (Word)

Returns or sets a  **Single** that represents the z-axis for the shape. Read/write.


## Syntax

 _expression_ . **Z**

 _expression_ An expression that returns a **[ThreeDFormat](threedformat-object-word.md)** object.
