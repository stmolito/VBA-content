# Bookmarks.Count Property (Word)

Returns the number of items in the  **Bookmarks** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Bookmarks](bookmarks-object-word.md)** object.
