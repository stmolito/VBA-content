# CalloutFormat.Type Property (Word)

Returns or sets the callout type. Read/write  **MsoCalloutType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[CalloutFormat](calloutformat-object-word.md)** object.
