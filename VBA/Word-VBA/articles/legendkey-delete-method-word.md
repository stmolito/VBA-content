# LegendKey.Delete Method (Word)

Deletes the object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **[LegendKey](legendkey-object-word.md)** object.


## Remarks

Deleting a  **LegendKey** object deletes the entire series.
