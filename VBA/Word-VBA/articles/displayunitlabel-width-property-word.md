# DisplayUnitLabel.Width Property (Word)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[DisplayUnitLabel](displayunitlabel-object-word.md)** object.
