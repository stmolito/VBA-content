# DataTable.Font Property (Word)

Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-word.md)** .


## Syntax

 _expression_ . **Font**

 _expression_ A variable that represents a **[DataTable](datatable-object-word.md)** object.
