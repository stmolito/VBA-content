# InlineShape.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **InlineShape** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[InlineShape](inlineshape-object-word.md)** object.
