# TableStyle.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TableStyle** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TableStyle](tablestyle-object-word.md)** object.
