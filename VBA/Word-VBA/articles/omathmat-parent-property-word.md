# OMathMat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathMat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathMat](omathmat-object-word.md)** object.
