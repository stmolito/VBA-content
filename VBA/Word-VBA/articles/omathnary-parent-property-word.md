# OMathNary.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathNary** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathNary](omathnary-object-word.md)** object.
