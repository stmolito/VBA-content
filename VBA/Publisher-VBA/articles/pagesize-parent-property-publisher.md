# PageSize.Parent Property (Publisher)

Returns the parent of the  **[PageSize](pagesizes-object-publisher.md)** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **PageSize** object.


### Return Value

Object
