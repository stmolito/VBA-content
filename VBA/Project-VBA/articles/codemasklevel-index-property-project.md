# CodeMaskLevel.Index Property (Project)

Gets the index of a  **CodeMaskLevel** object in the containing **CodeMask** collection. Read-only **Long**.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents a **CodeMaskLevel** object.


## Remarks

The  **Index** properties of other objects are used in similar ways. For an example, see the **[Index](project-index-property-project.md)** property of the **Project** object.
