# Windows2.Count Property (Project)

Gets the number of items in the  **Windows2** collection. Read-only **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **Windows2** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](assignments-count-property-project.md)** property.
