# ViewSingle.Index Property (Project)

Gets the index of a  **ViewSingle** object in the **ViewsSingle** containing object. Read-only **Long**.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents a **ViewSingle** object.


## Remarks

The  **Index** properties of different objects are used in similar ways. For an example, see the **[Index](project-index-property-project.md)** property of the **Project** object.
