# WorkWeekDay.Shift5 Property (Project)

Gets a  **[Shift](shift-object-project.md)** object representing the fifth work shift in a workweek day. Read-only **Shift**.


## Syntax

 _expression_. **Shift5**

 _expression_ An expression that returns a **WorkWeekDay** object.
