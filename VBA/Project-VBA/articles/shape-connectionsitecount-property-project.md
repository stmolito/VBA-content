# Shape.ConnectionSiteCount Property (Project)
Gets the number of connection sites on the shape. Read-only  **Long**.

## Syntax

 _expression_. **ConnectionSiteCount**

 _expression_ A variable that represents a **Shape** object.


## Property value

 **INT**
