# Shape.Table Property (Project)
Gets the  **ReportTable** object in the shape. Read-only[ReportTable](reporttable-object-project.md).

## Syntax

 _expression_. **Table**

 _expression_ A variable that represents a **Shape** object.


## Property value

 **REPORTTABLE**
