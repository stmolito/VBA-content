# Assignments.UniqueID Property (Project)

Gets an  **[Assignment](assignment-object-project.md)** object from its containing collection. Read-only **Assignment**.


## Syntax

 _expression_. **UniqueID**( ** _Index_** )

 _expression_ A variable that represents an **Assignments** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Long**|The position of the object within its containing collection.|
