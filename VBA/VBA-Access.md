
# Access VBA reference

In Access, programming is the process of adding functionality to your database by using Access macros or Visual Basic for Applications (VBA) code. This reference contains conceptual overviews, programming tasks, samples, and references to guide you in developing solutions based on Access:

- **Concepts**: Provides important concepts for developing custom Access solutions.
    
- **How-to Topics**: Provides step-by-step instructions for developing custom Access solutions.
    
- **Reference**: Provides reference materials for the Access object model.
